## Utilizar como Extensão

Este repositório pode ser adicionado como uma **extensão** no MakeCode.

* abrir [https://makecode.microbit.org/](https://makecode.microbit.org/)
* clica em **Novo Projeto**
* clica em **Extensões** sob o menu de definições
* procurar um **https://github.com/bredamatos/telecomando_recetor** e importar

## Editar este projeto ![Build status badge](https://github.com/bredamatos/telecomando_recetor/workflows/MakeCode/badge.svg)

Para editar este repositório no MakeCode.

* abrir [https://makecode.microbit.org/](https://makecode.microbit.org/)
* clica em **Importar** e, em seguida, em **Importar URL**
* colar **https://github.com/bredamatos/telecomando_recetor** e clicar em importar
